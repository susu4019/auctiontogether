# 프로젝트 소개
## 🏘️경매 투게더
부동산 경매 초보자들을 위해 부동산 경매 정보를 제공하는 서비스 애플리케이션.<br>
부동산 경매 물건 정보와 경매 관련 지식을 제공하고, 예상 낙찰가를 제공하여 부동산 경매 투자의 접근성을 높일 수 있음.<br><br><br><br>
